    Wallet Installation¶

    Installing the XLite wallet is a simple process. Below are step-by-step instructions for how to install the wallet on each operating system (OS). If you already have the XLite wallet installed and are updating, the steps are basically the same.

    Install GUI Wallet¶

    Windows

    1. Download the latest XLite wallet There are 3 options:

      1. XLite-x.x.x-win.exe (recommended - works on 64-bit and 32-bit Windows)
      2. XLite-x.x.x-win-x64.zip (for 64-bit Windows)
      3. XLite-x.x.x-win-ia32.zip (for 32-bit Windows)

      Note: Antivirus software may be flagged

      It is normal for antivirus software to flag the wallet as potentially dangerous. This is a common issue with most wallets, including Bitcoin's. As long as the wallet was downloaded from https://scalaris.info, Scalaris's Github it can be deemed as safe and the warning can be ignored. You may need to temporarily disable antivirus download protection to complete the download. You can read more about this issue here: https://bitcoin.org.

    2. Find the downloaded file. The default location is in Downloads.

    3. Before continuing, it is recommended to verify your download
    4. Continue to the instructions below for the download you selected (.exe or .zip):
    Important Tip: If installation fails to complete for either the .exe or .zip instructions below, click here for instructions what to do.
    1. Install Visual C++ 2010 Redistributable Package (x64).
    2. Redo the installation step below (.exe or .zip depending on your download).
    Using the .exe download (option a)

    1. Double-click the downloaded file to begin the installation process.

      Note: You may receive antivirus warning messages such as these.

      You may recieve warnings from antivirus software. These false positives are normal and can be dismissed.

      If you have Windows SmartScreen enabled, you may see the following message:

      Select More info, then Run anyway to dismiss this message:

    2. When the Setup Wizard appears, follow the setup instructions to finish installing XLite.

    3. When installation is complete, you can delete the downloaded .exe file.
    Using the .zip download (options b/c)
    1. Right-click the downloaded file, select Extract.
    2. After the files are extracted, a new folder should open with the contents. If a folder did not open, look for the new folder in the directory where the .zip file was downloaded.
    3. In the newly extracted folder you will find the XLite executable file.
    4. Move this XLite executable file to where you keep your applications, such as your Desktop. Additionally, you can right-click and Pin to Start or Pin to Taskbar for easy access.
    5. After moving this file, the .zip and the remaining extracted folders and files can be deleted since the XLite executable was the only file needed.
    MacOS

    1. Download the latest XLite wallet There is 1 option:

      • XLite-x.x.x-mac.dmg (recommended)

      Note: Antivirus software may be flagged

      It is normal for antivirus software to flag the wallet as potentially dangerous. This is a common issue with most wallets, including Bitcoin's. As long as the wallet was downloaded from https://scalaris.info, Scalaris's Github it can be deemed as safe and the warning can be ignored. You may need to temporarily disable antivirus download protection to complete the download. You can read more about this issue here: https://bitcoin.org.

    2. Find the downloaded file. The default location is in Downloads.

    3. Before continuing, it is recommended to verify your download
    4. Double-click the file to begin installation.
    5. Click and drag the XLite application icon over to the Applications folder and release.
    6. If you are prompted with a message asking if you would like to replace an existing version with a newer version, select Replace.
    7. Find the XLite installation drive on your desktop and eject it. (Right-click/two-finger tap on the XLite icon, then select Eject).
    Linux

    1. Download the latest XLite wallet There are 3 options:

      1. XLite-x.x.x-linux.AppImage (recommended - works on all flavors of Linux)
      2. XLite-x.x.x-linux.deb (Install package for Debian & Ubuntu)
      3. XLite-x.x.x-linux.tar.gz (Gzip archive - must be launched from Terminal)

      Note: Antivirus software may be flagged

      It is normal for antivirus software to flag the wallet as potentially dangerous. This is a common issue with most wallets, including Bitcoin's. As long as the wallet was downloaded from https://scalaris.info, Scalaris's Github it can be deemed as safe and the warning can be ignored. You may need to temporarily disable antivirus download protection to complete the download. You can read more about this issue here: https://bitcoin.org.

    2. You may be asked for a confirmation to download, select Save File then OK.

    3. Find the downloaded file. The default location is in Downloads.
    4. Before continuing, it is recommended to verify your download
    5. Continue to the instructions below for the download you selected:
    Using the .AppImage download (option a)
    1. Give executable permission to the .AppImage file as follows:
      1. Right-click the .AppImage file to display a dropdown menu like this:
      2. Select Properties at the bottom of the dropdown to reveal a screen similar to this:
      3. Select the Permissions tab and check the box, Allow executing file as program
      4. Click X in the upper left corner of the Properties window to close it
      5. Note: Alternatively, if you prefer the command line, you can simply use chmod u+x .AppImage-File to make the .AppImage file executable.
    2. Move the .AppImage file to where you keep your applications, such as your Desktop.
    Using the .deb download (option b)
    1. Double-click the downloaded file to open the installation screen.
    2. Select Install.
    3. You may be asked to enter your account password.
    4. The installation process may take a few minutes.
    Using the .tar.gz download (option c)
    1. Open a Linux Terminal.
    2. Type:tar -xzvf XLite-x.x.x-linux.tar.gz (replacing x.x.x with the version you downloaded).
    3. After files are extracted and unzipped by this command, you will see a new directory named XLite-x.x.x-linux.
    4. Move this new XLite-x.x.x-linux directory to where you keep your applications, such as your Desktop.

    Verifying Downloads¶

    It is important to verify the integrity of downloads before running them. Depending on how you downloaded it, it's possible the file may have been modified in transit to do something evil when run. The server hosting the download may also have been compromised.

    1. Get the sha256 hash of the release you download. These are provided on the Github release page as plain text in the release notes. The format follows SHA256-hash <filename>. Here is an example of the hashes: 
      95a25f5f903215b38ed5de7bb9f7b91caba6cd979c2100f5203d329bcd9d6bbe  XLite-1.0.0-linux.AppImage
1d5a42f2c23be44274708ff3b7816737edf3c6ba6a7f2f9907f3b6515c9f9916  XLite-1.0.0-linux.deb
7777496c75beeb944d2d65d128f22489f4e643da0aa691c457d359addc78b2b1  XLite-1.0.0-linux.tar.gz
a0a5135f19867ca6e7fddcced2f29f28281c809fdf192708cd7297511cd5567a  XLite-1.0.0-mac.dmg
f378be9cec6093ab84e0f2c1e4995d59ed2b9f43395af7530d43aa9d975b8502  XLite-1.0.0-mac.zip
1709c11a4e60483c3680158d4c34f6ba8716142e295c2b59ac56321630e982de  XLite-1.0.0-win-ia32.zip
5e1b97b5b8828a6a8a43a8dd9c89f40ee076baa1efee99d41b59558d1c25c07b  XLite-1.0.0-win-x64.zip
1c91238d192b7b4b8b7d32e54d4582ebb40f748228ed67f266fd24b929c2eb64  XLite-1.0.0-win.exe
    2. Take a note of the hash for the specific file you downloaded.

    3. Get the sha256 hash of the file you downloaded:

      Windows
      1. Open the command prompt.
      2. Navigate to the location of the downloaded file.
      3. Enter certUtil -hashfile filename SHA256 with filename replaced by the name fo the file you downloaded.

      Example: certUtil -hashfile xlite-1.0.0-win.exe SHA256

      MacOS
      1. Open the terminal.
      2. Navigate to the location of the downloaded file.
      3. Enter shasum -a 256 filename with filename replaced by the name fo the file you downloaded.

      Example: shasum -a 256 xlite-1.0.0-mac.dmg

      Linux
      1. Open the terminal.
      2. Navigate to the location of the downloaded file.
      3. Enter sha256sum filename with filename replaced by the name fo the file you downloaded.

      Example: sha256sum xlite-1.0.0-linux.AppImage

    4. Compare the release hash to the hash of the download. If the hashes do not match, DO NOT run the file and delete the file immediately.
