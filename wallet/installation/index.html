    Wallet Installation¶

    Installing the Scalaris wallet is a simple process. Choise wallet for your system and install.

    • For a complete guide on setting up the wallet, see the setup guide.
    Tip: There are two wallet variations.

    There are two wallet variations, a redesigned interface (default) and a classic interface. To use the classic wallet, you'll need to open your scalaris.conf file and add classic=1 on a new line (you may not have a scalaris.conf file and will need to create one). You can find the scalaris.conf file in your Scalaris data directory:

    1. Windows 
      C:\Users\[YourUsername]\AppData\Roaming\Scalaris\
      Or paste %appdata%\Roaming\Scalaris\ into the file explorer path field.
    2. MacOS 
      ~/Library/Application Support/Scalaris/
      Open Finder and in the program menu select Go > To Folder and enter the above path.
    3. Linux 
      ~/.scalaris/

    Install GUI Wallet¶

    Windows

    1. Download the latest Scalaris wallet. There are 4 options:

      1. scalaris-x.xx.x-win64-setup-unsigned.exe (recommended for 64-bit)
      2. scalaris-x.xx.x-win32-setup-unsigned.exe (recommended for 32-bit)
      3. scalaris-x.xx.x-win64.zip
      4. scalaris-x.xx.x-win32.zip

      Note: Antivirus software may be flagged

      It is normal for antivirus software to flag the wallet as potentially dangerous. This is a common issue with most wallets, including Bitcoin's. As long as the wallet was downloaded from https://scalaris.info, Scalaris's Github it can be deemed as safe and the warning can be ignored. You may need to temporarily disable antivirus download protection to complete the download. You can read more about this issue here: https://bitcoin.org.

    2. Before continuing, it is recommended to verify your download

    3. Continue to the instructions below for the download you selected:
    Using the .exe download (options a/b)
    1. Find the downloaded file. The default location is in Downloads.

    2. Double-click the file to begin the installation process.

      Note: You may receive antivirus warning messages.
      You may recieve warnings from antivirus software. These false positives are normal and can be dismissed.

      If you have Windows SmartScreen enabled, you may see the following message:

      Select More info, then Run anyway to dismiss this message:

    3. The installation guide will appear, select Next.

    4. Select Next to accept the default installation directory.

    5. Select Install to accept the default data directory folder name and begin installation.

    6. Select Next when the installation is complete.

    7. Select Finish to complete the installation process. If you would not like the wallet to open right away, uncheck the Run Scalaris option.

    Installation Complete!

    Continue to related guides:

    • Encrypt the wallet

    • Staking
    Using the .zip download (options c/d)
    1. Find the downloaded file. The default location is in Downloads.
    2. Right-click the file, select Extract All.
    3. Select Extract.
    4. After the files are extracted, a new folder should open with the contents. If a folder did not open, look for the new folder in the directory the downloaded file is located.
    5. In this new folder there will be another folder. Double-click on this sub-folder to view the contents.
    6. In this folder there will be a bin/ folder. Double-click on this sub-folder to view the contents.
    7. Here you will find the scalaris-qt executable file.
    8. Move this file you where you keep your applications, such as your Desktop. Additionally, you can right-click and Pin to Start or Pin to Taskbar for easy access.
    9. After moving this file, the files you downloaded can be deleted since the scalaris-qt executable file is all that is needed.

    10. Double-click the scalaris-qt file to begin the run wallet.

      Note: You may receive antivirus warning messages.
      You may recieve warnings from antivirus software. These false positives are normal and can be dismissed.

      If you have Windows SmartScreen enabled, you may see the following message:

      Select More info, then Run anyway to dismiss this message:

    Installation Complete!

    Continue to related guides:

    • Encrypt the wallet

    • Scalaris DX setup
    • Staking
    MacOS

    1. Download the latest Scalaris wallet. There are 3 options:

      1. scalaris-x.xx.x-osx-unsigned.dmg (recommended)
      2. scalaris-x.xx.x-osx-unsigned.tar.gz
      3. scalaris-x.xx.x-osx64.tar.gz

      Note: Antivirus software may be flagged

      It is normal for antivirus software to flag the wallet as potentially dangerous. This is a common issue with most wallets, including Bitcoin's. As long as the wallet was downloaded from https://scalaris.info, Scalaris's Github it can be deemed as safe and the warning can be ignored. You may need to temporarily disable antivirus download protection to complete the download. You can read more about this issue here: https://bitcoin.org.

    2. Before continuing, it is recommended to verify your download

    3. Continue to the instructions below for the download you selected:
    Using the osx-unsigned.dmg download (option a)
    1. Find the downloaded file. The default location is in Downloads.
    2. Double-click the file to begin installation.
    3. Click and drag the Scalaris-Qt application icon over to the Applications folder and release.
    4. Open Finder, navigate to Applications, and find Scalaris-Qt in the list of applications.
    5. Right-click the file and select Open. If using the touch pad this can be done by clicking with 2 fingers.
    6. If you are prompted with a message asking if you are sure you want to open the application, select Open.
    7. Your computer will begin verifying the application. This may take a few minutes to complete.
    8. An installation prompt will appear, select OK.
    9. Run scalaris wallet.
    Installation Complete!

    Continue to related guides:

    • Sync the wallet
    • Encrypt the wallet
    • Lock/unlock wallet
    • Scalaris DX setup
    • Staking
    Using the osx-unsigned.tar.gz download (option b)
    1. Find the downloaded file. The default location is in Downloads.
    2. Double-click the file to unpack the contents.
    3. In the newly unpacked folder, navigate to the dist folder.
    4. Click and drag the Scalaris-Qt application over to the Applications folder and release. This will add the wallet to Applications.
    5. If you are prompted with a message asking if you would like to replace an existing version with a newer version, select Replace.
    6. The downloaded files may now be removed.
    7. Open Finder, navigate to Applications, and find Scalaris-Qt in the list of applications.
    8. Right-click the file and select Open. If using the touch pad this can be done by clicking with 2 fingers.
    9. If you are prompted with a message asking if you are sure you want to open the application, select Open.
    10. Your computer will begin verifying the application. This may take a few minutes to complete.
    11. An installation prompt will appear, select OK.
    12. Run scalaris wallet.
    Installation Complete!

    Continue to related guides:

    • Sync the wallet
    • Encrypt the wallet
    • Lock/unlock wallet
    • Scalaris DX setup
    • Staking
    Using the osx64.tar.gz download (option c)
    1. Find the downloaded file. The default location is in Downloads.
    2. Double-click the file to unpack the contents.
    3. In the new folder created, navigate to the bin/ folder.
    4. Double-click the scalaris-qt file.
    5. If you are prompted with a message asking if you are sure you want to open the application, select Open.
    6. Your computer will begin verifying the application. This may take a few minutes to complete.
    7. Since this is a command line executable file, a command-line window will appear to start the wallet, which begins the installation process.
    8. An installation prompt will appear, select OK.
    9. Run scalaris wallet.
    Installation Complete!

    Linux

    1. Download the latest Scalaris wallet. There are 3 options:

      1. scalaris-x.xx.x-x86_64-linux-gnu.tar.gz (recommended for 64-bit)
      2. scalaris-x.xx.x-i686-pc-linux-gnu.tar.gz (recommended for 32-bit)
      3. scalaris-x.xx.x-arm-linux-gnueabihf.tar.gz (recommended for Raspberry Pi)

      Note: Antivirus software may be flagged

      It is normal for antivirus software to flag the wallet as potentially dangerous. This is a common issue with most wallets, including Bitcoin's. As long as the wallet was downloaded from https://scalaris.info, Scalaris's Github it can be deemed as safe and the warning can be ignored. You may need to temporarily disable antivirus download protection to complete the download. You can read more about this issue here: https://bitcoin.org.

    2. You may be asked for a confirmation to download, select Save File then OK.

    3. Find the downloaded file. The default location is in Downloads.
    4. Before continuing, it is recommended to verify your download
    5. Right-click the file, select Extract Here.
    6. Double-click the scalaris-x.xx.x-x86_64-linux-gnu/ folder to view the contents.
    7. Double-click the scalaris-x.xx.x/ folder to view the contents.
    8. Double-click the bin/ folder to view the contents.
    9. Here you will find the scalaris-qt executable file.
    10. Move this file you where you keep your applications, such as your Desktop.
    11. After moving this file, the files you downloaded can be deleted since the scalaris-qt executable file is all that is needed.
    12. Double-click the scalaris-qt file to begin the installation process. (If it fails to open on double-click, you may need to give executable permission to the file: Either sudo chmod +x scalaris-qt in Debian terminal, or Right-click property, give executable permission.)
    13. An installation prompt will appear, select OK.
    14. Run scalaris wallet.
    Installation Complete!

    Verifying Downloads¶

    It is important to verify the integrity of downloads before running them. Depending on how you downloaded it, it's possible the file may have been modified in transit to do something evil when run. The server hosting the download may also have been compromised.

    1. Get the sha256 hash of the release you download. These are provided on the Github release page either in a file called sha256sums.txt or as a plain text in the release notes. The format follows SHA256(filename)= hash. Here is an example of the hashes: 
      SHA256(scalaris-1.0.0-arm-linux-gnueabihf-debug.tar.gz)= 77e6c3e5d1e6ef9f0d54a8f0fe3d638b1c6bd889e609c6a69bb8f82322117847
SHA256(scalaris-1.0.0-arm-linux-gnueabihf.tar.gz)= 94705029ebf05b0aa2b0939dd0da0663e578accbf5dca8e9393456d9d40edde6
SHA256(scalaris-1.0.0-i686-pc-linux-gnu-debug.tar.gz)= 30039c5eb3951a73d578f613d072a79610cb3983113cf816fbed485ce44bd57c
SHA256(scalaris-1.0.0-i686-pc-linux-gnu.tar.gz)= c5c9013b5e0539a99825339f98a460281014b3dca5d970e24faa860b6fd0c0e7
SHA256(scalaris-1.0.0-osx-unsigned.tar.gz)= 0d5be3dedc83521349b5def54c1f045f1cd35c67d8da5beb50c90beaa50ae659
SHA256(scalaris-1.0.0-osx64.tar.gz)= 438b0b82001ca5ce0af8300061207f87421c3c9cfeab47dd4c56ebc856b357da
SHA256(scalaris-1.0.0-win-unsigned.tar.gz)= 1f81058be74d3714f1e07f5b16562f471cd2da1fac8251b3c3d414dba5b0d271
SHA256(scalaris-1.0.0-win32-debug.zip)= fdc15df95dbb1293d59a2a9b90bf421594b0dc27320cd09cf31dcc9073bb2a8e
SHA256(scalaris-1.0.0-win32-setup-unsigned.exe)= c89200719d3e55f9f7eee1cf8f77a4a4c43d8c4551b896a4f0d7617a935f31e0
SHA256(scalaris-1.0.0-win32.zip)= 60c0805f00fe4e1c1e93af2ef3e93fb0fbe3b7bcf2385f8192c1a176b4068c3f
SHA256(scalaris-1.0.0-win64-debug.zip)= e739bf19e5d826ff9750c9ae9dfddf0bf17b9e6da2d90e094698c02171454a6f
SHA256(scalaris-1.0.0-win64-setup-unsigned.exe)= 350d67cdb009a7b4d3101dae145caa9c69e8c7a2ca6252a4d97aedf33a5a3f9d
SHA256(scalaris-1.0.0-win64.zip)= f1f472353f034e1151345c167a48a01d1ceafdd2b7a5b7a5cc82df9cbef6ce6d
SHA256(scalaris-1.0.0-x86_64-linux-gnu-debug.tar.gz)= 998fa0487cc4e8fc2be9fb004cd6eb9bc0e91d690da110a5073e3f3b50bd2938
SHA256(scalaris-1.0.0-x86_64-linux-gnu.tar.gz)= 9608a66feb4616092ddb2dbeeafef01f5c0c28b61cccf13bcd032cee3981be74
    2. Take a note of the hash for the specific file you downloaded.

    3. Get the sha256 hash of the file you downloaded:

      Windows
      1. Open the command prompt.
      2. Navigate to the location of the downloaded file.
      3. Enter certUtil -hashfile filename SHA256 with filename replaced by the name fo the file you downloaded.

      Example: certUtil -hashfile scalaris-1.0.0-win64.zip SHA256

      MacOS
      1. Open the terminal.
      2. Navigate to the location of the downloaded file.
      3. Enter shasum -a 256 filename with filename replaced by the name fo the file you downloaded.

      Example: shasum -a 256 scalaris-1.0.0-osx-unsigned.tar.gz

      Linux
      1. Open the terminal.
      2. Navigate to the location of the downloaded file.
      3. Enter sha256sum filename with filename replaced by the name fo the file you downloaded.

      Example: sha256sum scalaris-1.0.0-x86_64-linux-gnu.tar.gz

    4. Compare the release hash to the hash of the download. If the hashes do not match, DO NOT run the file and delete the file immediately.
